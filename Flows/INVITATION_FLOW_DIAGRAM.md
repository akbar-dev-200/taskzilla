## Quick Reference Card

**Send Invitation:**
```
POST /api/invites
Auth: Required (Team Admin)
Body: { team_id, invitations: [{ email, role }] }
```

**Accept Invitation:**
```
POST /api/invites/accept
Auth: Required
Body: { token }
```

**Get My Invitations:**
```
GET /api/invites/my-pending
Auth: Required
```

**Revoke Invitation:**
```
DELETE /api/invites/{id}
Auth: Required (Inviter)
```
